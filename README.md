# Project 3 SpecKit Starter

This folder mirrors the repository skeleton we will seed in GitHub Classroom.

Key commands (run from repo root):

```bash
uv run python tools/validate_manifest.py --path project3.yaml --check-paths
speckit check --report reports/speckit_report.json
pytest tests/redbar --maxfail=1
```

Replace the placeholder files before submitting. Each directory is pre-created so CI and local tooling work out of the box.
